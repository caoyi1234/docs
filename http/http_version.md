## HTTP/0.9

+ 只有一个命令GET
+ 没有header（请求头和响应头）等描述数据的信息
+ 服务器发送完毕，就关闭TCP连接

## HTTP/1.0

+ 增加了很多命令，例如POST、PUT等
+ 增加status code（状态码）和header（请求头和响应头）
+ 多字符集支持、多部分发送、权限、缓存等

## HTTP/1.1

+ 持久连接
+ pipeline
+ 增加host和其他一些命令

## HTTP/2.0

+ 所有数据以二进制传输（http1.1中大部分数据是以字符串传输）
+ 同一个连接里面发送多个请求不再需要按照顺序来
+ 头信息压缩以及推送等提高效率的功能
