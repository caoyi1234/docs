## electron init vue
> vue add electron-builder 安装electron-builder依赖
1. 更行package.json中的脚本文件，添加main:background.js
2. src目录下增加background.js文件
3. 期间会新增选择electron的版本
4. 运行npm run electron:serve 构建项目，可以实时编译

> 主进程：background.js
